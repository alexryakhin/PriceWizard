Price Wizard ตั้งราคาสมาชิก App Store สำหรับทุกเขตจากหน้าจอเดียว - ไม่ต้องอัปเดต 175 ประเทศด้วยตัวเองอีกต่อไป

สำหรับนักพัฒนาที่มีสมาชิก

• เชื่อมต่อด้วยคีย์ API App Store Connect (Key ID, Issuer ID, ไฟล์ .p8) คีย์อยู่เฉพาะใน Keychain ของ Mac ของคุณ เราไม่เคยเห็น
• เลือกราคาฐานสหรัฐ แล้วดูตัวอย่างเต็ม: ราคาใหม่ต่อเขต เป็นสกุลเงินท้องถิ่นและ USD
• เลือกราคาตามภูมิภาคแบบ Apple Equalization, Netflix หรือ Spotify แล้วปรับแต่ละเขตได้
• นำการเปลี่ยนแปลงทั้งหมดไปใช้ครั้งเดียว แอปจะข้ามเขตที่มีราคาเป้าหมายแล้ว และไม่อนุญาตให้ใช้กับวันที่มีการเปลี่ยนแปลงตามกำหนดแล้ว

ไม่ต้องมีบัญชีกับเรา - แค่ Apple ID และคีย์ API App Store Connect ของคุณ

ความเป็นส่วนตัว

เราไม่เก็บหรือรับข้อมูลของคุณ ข้อมูลเข้าสู่ระบบเก็บเฉพาะใน Keychain ของ Mac แอปสื่อสารเฉพาะกับ API ของ Apple และบริการอัตราแลกเปลี่ยนสาธารณะสำหรับแปลงเป็น USD

ต้องใช้: macOS 15 ขึ้นไป คีย์ API App Store Connect ที่มีสิทธิ์ App Manager (หรือ Admin)
