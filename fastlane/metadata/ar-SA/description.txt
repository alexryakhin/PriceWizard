يُحدّث Price Wizard أسعار اشتراكات App Store لجميع المناطق من شاشة واحدة - دون تحديث 175 دولة يدوياً.

لمطوّري التطبيقات مع اشتراكات

• الاتصال بمفتاح API لـ App Store Connect (Key ID وIssuer ID وملف .p8). يبقى المفتاح في Keychain على جهاز Mac؛ نحن لا نراه.
• اختر سعراً أساسياً بالدولار الأمريكي واطّلع على المعاينة الكاملة: السعر الجديد لكل منطقة بالعملة المحلية وبالدولار.
• اختر تسعيراً إقليمياً بأسلوب Apple Equalization أو Netflix أو Spotify وعدّل أي منطقة.
• طبّق كل التغييرات دفعة واحدة. التطبيق يتخطى المناطق التي بلغت السعر المستهدف ولا يسمح بالتطبيق في تاريخ فيه تغيير مجدول.

لا حساب لدينا - فقط Apple ID ومفتاح API لـ App Store Connect.

الخصوصية

لا نجمع ولا نستلم بياناتك. تُخزَّن بيانات الاعتماد فقط في Keychain على جهاز Mac. التطبيق يتواصل فقط مع واجهة Apple ومع خدمة صرف عامة لتحويل الدولار.

المتطلبات: macOS 15 أو أحدث. مفتاح API لـ App Store Connect بصلاحية App Manager (أو Admin).
