Price Wizard מגדיר את מחירי מנויי App Store לכל האזורים ממסך אחד - בלי לעדכן 175 מדינות ידנית.

למפתחים עם מנויים

• התחבר עם מפתח API של App Store Connect (Key ID, Issuer ID, קובץ .p8). המפתח נשאר ב-Keychain של ה-Mac; אנחנו לא רואים אותו.
• בחר מחיר בסיס בארה"ב וצפה בתצוגה מלאה: מחיר חדש לאזור במטבע מקומי ו-USD.
• בחר תמחור אזורי בסגנון Apple Equalization, Netflix או Spotify והתאם כל אזור.
• החל את כל השינויים בבת אחת. האפליקציה מדלגת על אזורים עם המחיר היעד ולא מאפשרת להחיל בתאריך עם שינוי מתוזמן.

אין חשבון אצלנו - רק Apple ID ומפתח API של App Store Connect.

פרטיות

אנחנו לא אוספים ולא מקבלים את הנתונים שלך. פרטי ההתחברות נשמרים רק ב-Keychain של ה-Mac. האפליקציה מתקשרת רק עם ה-API של Apple ושירות שערי חליפין ציבורי להמרה ל-USD.

נדרש: macOS 15 ומעלה. מפתח API של App Store Connect עם הרשאת App Manager (או Admin).
