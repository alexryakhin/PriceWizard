Price Wizard задаёт цены подписок App Store для всех регионов с одного экрана - больше не нужно вручную обновлять 175 стран.

ДЛЯ РАЗРАБОТЧИКОВ С ПОДПИСКАМИ

• Подключитесь ключом API App Store Connect (Key ID, Issuer ID, файл .p8). Ключ хранится в связке ключей Mac; мы его не видим.
• Выберите базовую цену в США и смотрите полный предпросмотр: новая цена по каждому региону в местной валюте и в долларах.
• Выберите региональное ценообразование: Apple Equalization, в стиле Netflix или Spotify и при необходимости измените любой регион.
• Примените все изменения одним действием. Приложение пропускает регионы с уже нужной ценой и не даёт применить изменения на дату с уже запланированным изменением.

Никакого аккаунта у нас — только ваш Apple ID и ключ API App Store Connect.

КОНФИДЕНЦИАЛЬНОСТЬ

Мы не собираем и не получаем ваши данные. Учётные данные хранятся только в связке ключей Mac. Приложение обращается только к API Apple и к публичному сервису курсов валют для пересчёта в доллары.

Требования: macOS 15 или новее. Ключ API App Store Connect с доступом App Manager (или Admin).
