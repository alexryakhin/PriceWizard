Price Wizard एक ही स्क्रीन से हर टेरिटरी के लिए App Store सब्सक्रिप्शन कीमतें सेट करता है — 175 देश हाथ से अपडेट करने की ज़रूरत नहीं।

सब्सक्रिप्शन वाले डेवलपर्स के लिए

• अपने App Store Connect API कुंजी (Key ID, Issuer ID, .p8 फ़ाइल) से कनेक्ट करें। कुंजी आपके Mac के Keychain में रहती है; हम इसे कभी नहीं देखते।
• US बेस कीमत चुनें और पूरा प्रीव्यू देखें: टेरिटरी के हिसाब से नई कीमत, स्थानीय मुद्रा और USD में।
• Apple Equalization, Netflix-स्टाइल या Spotify-स्टाइल रीजनल प्राइसिंग चुनें और किसी भी टेरिटरी को ट्वीक करें।
• सारे बदलाव एक साथ लागू करें। ऐप उन टेरिटरी को छोड़ देता है जिन पर पहले से टारगेट कीमत है और उस तारीख पर लागू नहीं होने देता जिस पर पहले से बदलाव शेड्यूल है।

हमारे साथ कोई अकाउंट नहीं — बस आपका Apple ID और App Store Connect API कुंजी।

प्राइवेसी

हम आपका डेटा कलेक्ट या रिसीव नहीं करते। क्रेडेंशियल्स सिर्फ आपके Mac के Keychain में स्टोर होते हैं। ऐप सिर्फ Apple के API और USD कन्वर्जन के लिए पब्लिक एक्सचेंज-रेट सर्विस से बात करता है।

ज़रूरी: macOS 15 या बाद। App Manager (या Admin) एक्सेस वाली App Store Connect API कुंजी।
