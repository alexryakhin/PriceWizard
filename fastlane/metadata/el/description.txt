Το Price Wizard ορίζει τις τιμές συνδρομών App Store για όλες τις περιοχές από μία οθόνη - χωρίς χειροκίνητη ενημέρωση 175 χωρών.

ΓΙΑ ΠΡΟΓΡΑΜΜΑΤΙΣΤΕΣ ΜΕ ΣΥΝΔΡΟΜΕΣ

• Σύνδεση με το κλειδί API App Store Connect (Key ID, Issuer ID, αρχείο .p8). Το κλειδί μένει στο Keychain του Mac σας· εμείς δεν το βλέπουμε ποτέ.
• Επιλέξτε βασική τιμή USD και δείτε πλήρη προεπισκόπηση: νέα τιμή ανά περιοχή σε τοπικό νόμισμα και USD.
• Επιλέξτε τοπικές τιμές στυλ Apple Equalization, Netflix ή Spotify και προσαρμόστε οποιαδήποτε περιοχή.
• Εφαρμόστε όλες τις αλλαγές με μία κίνηση. Η εφαρμογή παραλείπει περιοχές που έχουν ήδη την τιμή-στόχο και δεν επιτρέπει εφαρμογή σε ημερομηνία με ήδη προγραμματισμένη αλλαγή.

Χωρίς λογαριασμό σε εμάς - μόνο το Apple ID και το κλειδί API App Store Connect σας.

ΑΠΟΡΡΗΤΟ

Δεν συλλέγουμε ούτε λαμβάνουμε τα δεδομένα σας. Τα στοιχεία σας αποθηκεύονται μόνο στο Keychain του Mac σας. Η εφαρμογή επικοινωνεί μόνο με το API της Apple και μια δημόσια υπηρεσία συναλλάγματος για μετατροπή σε USD.

Απαιτήσεις: macOS 15 ή νεότερο. Κλειδί API App Store Connect με πρόσβαση App Manager (ή Admin).
