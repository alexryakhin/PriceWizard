Price Wizard задає ціни підписок App Store для всіх регіонів з одного екрана — більше не потрібно вручну оновлювати 175 країн.

ДЛЯ РОЗРОБНИКІВ ІЗ ПІДПИСКАМИ

• Підключіться ключем API App Store Connect (Key ID, Issuer ID, файл .p8). Ключ зберігається в Keychain вашого Mac; ми його не бачимо.
• Виберіть базову ціну в США та перегляньте повну передперегляд: нова ціна по кожному регіону в місцевій валюті та в доларах.
• Оберіть регіональне ціноутворення: Apple Equalization, у стилі Netflix або Spotify та за потреби змініть будь-який регіон.
• Застосуйте всі зміни одним дією. Додаток пропускає регіони з уже потрібною ціною та не дозволяє застосувати зміни на дату з уже запланованою зміною.

Без облікового запису у нас — лише ваш Apple ID та ключ API App Store Connect.

КОНФІДЕНЦІЙНІСТЬ

Ми не збираємо та не отримуємо ваші дані. Облікові дані зберігаються лише в Keychain вашого Mac. Додаток звертається лише до API Apple та до публічного сервісу курсів для перерахунку в долари.

Вимоги: macOS 15 або новіший. Ключ API App Store Connect з доступом App Manager (або Admin).
